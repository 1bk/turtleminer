# turtleminer
Mining TRTLs

## Setup
```
git clone https://github.com/mavx/turtleminer
cd turtleminer
bash setup_trtl.sh
```

## Usage
```
cd turtleminer
bash mine.sh
```

## Config
- Please edit your pools and TRTL address in `pools.txt`, otherwise thank you very much :)
